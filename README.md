# 🎈 Happy Birthday

同僚の誕生日を祝うための3Dバルーンアニメーションページです。

## デモ

🔗 **[https://wataru-fujita.github.io/happy-birthday/](https://wataru-fujita.github.io/happy-birthday/)**

> ※ リポジトリ名が異なる場合は上記URLの `happy-birthday` 部分を変更してください。

## 特徴

- **Babylon.js** による3D表示（CDN読み込み、ビルド不要）
- 8個のバルーンが輪を描きながら螺旋状に上昇するアニメーション
- マウス操作で自由に視点変更
- 星のパーティクルが漂う背景
- `index.html` 1ファイルのみで動作

## カラー

思い出の色をバルーンに込めました 🎨

| カラー | カラーコード |

| 🩷 マゼンタ | `#FF00FF` |
| 💛 イエロー | `#FFFF00` |
| 💚 グリーン | `#00FF00` |
| 🩶 グレー | `#808080` |

## 操作方法

| 操作 | 動作 |
|------|------|
| 左ドラッグ | 視点を回転 |
| マウスホイール | ズームイン / アウト |
| 右ドラッグ | 視点をパン（平行移動） |

## 使用技術

- HTML / JavaScript（vanilla）
- [Babylon.js](https://www.babylonjs.com/)（CDN）

## 公開方法（GitHub Pages）

1. このリポジトリを GitHub に push
2. Settings → Pages → Branch を `main`、フォルダを `/ (root)` に設定
3. `https://<username>.github.io/<repo-name>/` でアクセス
